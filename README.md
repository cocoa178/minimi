# minimi

나만의 프라이빗 기록 앱

## 🌟 특징

- **100% 프라이빗**: 모든 데이터는 브라우저의 localStorage에만 저장됩니다
- **서버 전송 없음**: 어떤 데이터도 외부로 전송되지 않습니다
- **간단한 기록**: 한 줄 메모부터 시작
- **AI 분석**: Claude에게 분석 요청할 수 있는 프롬프트 자동 생성

## 📱 기능

### 홈
- 오늘의 감정 기록 (5단계)
- 빠른 메모 작성 (태그 지정 가능)
- 오늘의 통계

### 아카이브
- 책, 영화, TV, 음악, 장소 기록
- 카테고리별 관리

### 리포터 M
- 일간/주간/월간 통계
- 감정 평균, 태그 분석
- **Claude AI 분석 프롬프트 생성**

### 설정
- 데이터 백업 (JSON 내보내기)
- 데이터 복원 (JSON 가져오기)
- 전체 삭제

## 🚀 GitHub Pages 배포 방법

1. 이 저장소를 Fork 하거나 파일들을 새 저장소에 업로드
2. 저장소 Settings > Pages로 이동
3. Source를 "Deploy from a branch"로 선택
4. Branch를 "main" (또는 master), 폴더를 "/ (root)"로 선택
5. Save 클릭
6. 몇 분 후 `https://[username].github.io/[repo-name]`에서 접속 가능

## 💡 사용법

1. 웹사이트 접속
2. 홈 화면에서 감정과 메모 기록
3. 아카이브에서 본 책/영화 등 기록
4. 리포터 M에서 통계 확인
5. "프롬프트 생성하기" 버튼으로 Claude 분석용 텍스트 복사
6. [Claude.ai](https://claude.ai)에 붙여넣기하여 분석 받기

## 🔒 프라이버시

- 모든 데이터는 사용자의 브라우저에만 저장됩니다
- 서버로 전송되는 데이터가 없습니다
- Claude 분석 시에도 사용자가 직접 복사-붙여넣기 방식으로 전달합니다
- 백업 파일도 사용자의 기기에만 저장됩니다
